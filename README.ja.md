# 🌌 Astro-Yama: Geek's Bento Universe

**Astro 6** で構築された、独創的なギークスタイルの個人サイトです。現代的な Bento UI の美学とレトロなギーク・スピリットを融合させた、ユニークなデジタル・プレイグラウンドです。

![Homepage Preview](./public/images/screenshots/homepage.png)

## 🎨 コア・デザイン哲学

このサイトの本質は **「インタラクティブな美学」** にあります。単なる静的なレイアウトを排し、緻密に設計されたマイクロ・インタラクション、3D ビジュアル、ピクセルアートを通じて、訪問者に特別な探索体験を提供します。

### 🌟 主な機能

*   **Bento Grid ダッシュボード**: 現代的でモジュール化されたレイアウト。あらゆる画面にフィットし、コアなエントリーポイントをエレガントに集約します。
*   **3D 惑星軌道「友だちリンク」**: 従来のリスト形式とはおさらばです。私を中心に、友だちのアイコンが衛星のように 360° の軌道を描いて回る重力システムを採用しています。
*   **ゲーム・ラボ (Game Lab)**: 倉庫番 (Sokoban) などのミニゲームを内蔵。キーボードとタッチ操作の両方に対応し、Web 技術の可能性を提示します。
*   **シグマ (Sigma) プロフィール**: ZH/EN/JA の 3 言語に対応した高度な多言語ソリューション。シームレスな切り替えと、シャープな「シグマ男子」スタイルのビジュアル。
*   **ギークなこだわりアニメーション**:
    *   **グリッド・キャタピラー**: Bento の縁を這うコードの虫。
    *   **ピクセルキャット「トップへ戻る」**: スクロールバーで休んでいる黒猫。クリックすると空へ駆け上がり、一瞬でトップへ戻ります。
    *   **バブル・フラッド・マスク**: ページ読み込み時に水が溢れ、泡が昇るユニークなスプラッシュ画面。

## 🛠️ 技術スタック

*   **フレームワーク**: [Astro 6](https://astro.build/) (Island Architecture)
*   **UI コンポーネント**: [React 19](https://reactjs.org/)
*   **スタイリング**: [Tailwind CSS 4](https://tailwindcss.com/)
*   **アニメーション**: [Framer Motion](https://www.framer.com/motion/)
*   **状態管理**: [Nanostores](https://github.com/nanostores/nanostores)
*   **アイコン**: [Lucide React](https://lucide.dev/)

## 🚀 クイックスタート

### 1. クローンとインストール
```bash
git clone https://github.com/your-username/astro-yama.git
cd astro-yama
pnpm install
```

### 2. 開発サーバーの起動
```bash
pnpm dev
```
`http://localhost:4321` でプレビュー可能です。

### 3. 本番用ビルド
```bash
pnpm build
```
