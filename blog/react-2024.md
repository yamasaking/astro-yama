# React 2024 生态观察

React 依然是全球最受欢迎的前端框架之一。随着 2024 年的到来，我们看到了 React 团队对于“编译器”和“服务器组件”的深度思考。

## 核心趋势

### 1. React Server Components (RSC)
RSC 正在改变我们组织数据获取的方式。通过在服务器上运行组件，我们可以显著减少发送到客户端的 JavaScript 体积。

### 2. React Forget (Compiler)
自动化记忆化 (memoization) 是社区期待已久的特性。它旨在让开发者不再需要手动编写 `useMemo` 和 `useCallback`。

## 开发者该如何准备？

学习如何利用 Next.js 等框架提供的 App Router，以及理解 RSC 的工作原理，将成为每个 React 开发者的必修课。
