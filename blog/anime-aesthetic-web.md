# 二次元美学

在高度标准化的 Web 设计中，引入一些动漫元素往往能给用户带来意想不到的惊喜与情感连接。

## 像素艺术的回归

从 FC 时代的点阵风到现代高保真像素画，这种独特的形式既带有怀旧色彩，又充满了数字艺术的可塑性。

- **低延迟动效**: 极简的位移体现生动感。
- **高饱和配色**: 打造极具视觉冲击力的界面。

将动漫的活力融入 UI，不仅是样式的改变，更是一种个性的表达。
