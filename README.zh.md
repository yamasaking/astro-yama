# 🌌 Astro-Yama: Geek's Bento Universe

一个基于 **Astro 6** 构建的、极具个性的极客风个人站点。它不仅是一个网站，更是一个融合了现代 Bento 美学与复古极客精神的数字实验场。

![Homepage Preview](./public/images/screenshots/homepage.png)

## 🎨 核心设计理念

本站点的核心在于 **“交互式美学”**。我们拒绝平庸的静态布局，通过精心设计的微交互、3D 视觉以及像素艺术，为访客提供一段独特的探索旅程。

### 🌟 亮点功能

*   **Bento Grid 仪表盘**: 现代化的模块化布局，完美适配各种屏幕，将核心入口优雅聚合。
*   **3D 星球环绕友链**: 告别传统的列表式友链。这是一个以我为中心的引力系统，好友头像化身卫星，依照 360° 轨道实时环绕。
*   **游戏实验室 (Game Lab)**: 内置交互式小游戏（如推箱子 Sokoban），支持键盘与触控操作，展现 Web 技术的无限可能。
*   **西格玛 (Sigma) 个人简介**: 深度集成的多语言方案 (ZH/EN/JA)，支持一键无感切换，配合冷峻的西格玛风格视觉。
*   **极客细节动效**:
    *   **网格毛毛虫**: 在 Bento 边缘爬行的代码虫。
    *   **像素猫回到顶部**: 一只趴在滚动条上的像素黑猫，伴随您的点击冲向云端。
    *   **泡泡水淹遮罩**: 独特的页面渲染启动画面，水光涟漪间开启访问体验。

## 🛠️ 技术栈

*   **框架**: [Astro 6](https://astro.build/) (Island Architecture)
*   **逻辑组件**: [React 19](https://reactjs.org/)
*   **样式**: [Tailwind CSS 4](https://tailwindcss.com/)
*   **动画**: [Framer Motion](https://www.framer.com/motion/)
*   **状态管理**: [Nanostores](https://github.com/nanostores/nanostores)
*   **图标**: [Lucide React](https://lucide.dev/)

## 🚀 快速开始

### 1. 克隆并安装
```bash
git clone https://github.com/your-username/astro-yama.git
cd astro-yama
pnpm install
```

### 2. 启动开发服务器
```bash
pnpm dev
```
访问 `http://localhost:4321` 即可预览。

### 3. 构建生产环境
```bash
pnpm build
```
